Parasol Example
===============

`Parasol <http://parasol.readthedocs.io/en/latest/>`_ is a Framework for Parametric Model Investigation.


See the Docs at: `<http://parasol.readthedocs.org/en/latest/>`_

`Parasol <http://parasol.readthedocs.io/en/latest/>`_  enables easy optimization, sensitivity study, and visualization.

For this example, `Parasol <http://parasol.readthedocs.io/en/latest/>`_ was used along with RocketCEA
to evaluate sea level performance of a rocket engine.
